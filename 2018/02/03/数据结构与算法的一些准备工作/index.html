int* copyIntArray(int a[], int n){
    int* arr = new int[n];
    copy(a,a+n,arr);
    return arr;
}

以上我们的数组是采用new的方式来开辟内存空间的，所以在使用完了之后需要delete，虽然不delete并不会影响什么，但你应该去养成这样的一个好习惯。
